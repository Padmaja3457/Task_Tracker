# 🚀 Task Tracker - MERN Stack

A powerful and lightweight task management application built with the MERN stack. This project demonstrates full-stack integration, featuring a **Vite-powered React frontend** and a **Node.js/Express backend** connected to **MongoDB Atlas**.

---

## 🔗 Live Demo
Check out the live application here:  
👉 **https://task-tracker-seven-self.vercel.app/** 

---

## 🛠 Tech Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend** | React.js (Vite), Axios, CSS3 |
| **Backend** | Node.js, Express.js |
| **Database** | MongoDB|
| **Deployment** | Vercel (Frontend) & Railway (Backend) |

---

## ✨ Features

*   **Full CRUD Functionality:** Create, view, and delete tasks seamlessly.
*   **Production Ready:** Specifically configured for deployment with `0.0.0.0` host binding.
*   **Fast Builds:** Optimized with Vite for a superior development experience.
*   **Database Integration:** Persistent storage using MongoDB cloud clusters.

## 🚀 Installation & Local Setup

To run this project locally, follow these steps:

### 1. Clone the Repository
```bash
git clone https://github.com/Padmaja3457/Task_Tracker.git
cd your-repo-name
```
## 2. Backend Setup
```bash
# Navigate to server directory
cd server

# Install dependencies
npm install

# Start the server
npm start
```
## 3. Frontend Setup
```bash
# Navigate to client directory
cd ../client

# Install dependencies
npm install

# Run in development mode
npm run dev
```
